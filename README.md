# homebrew-aliast

Custom Homebrew tap for [AliasT](https://github.com/ChrisWoo0443/AliasT) — ghost-text autocompletion and natural-language commands for zsh on macOS.

## Install

```bash
brew tap ChrisWoo0443/aliast
brew install aliast
```

After install, follow the post-install caveats to source the plugin in your `.zshrc`.
